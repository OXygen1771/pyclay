# Python bindings for `clay.h`

Python bindings for [Clay](https://github.com/nicbarker/clay/), a
high-performance UI layout library written in C.

This project is in *very early* development, so much so that there isn't any
documentation. And there probably won't be much, clay concepts and API details
will still apply, so, if you haven't, take a look at
[clay's documentation](https://github.com/nicbarker/clay?tab=readme-ov-file#high-level-documentation).
